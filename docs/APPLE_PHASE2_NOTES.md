# Phase 2 — Apple UX Polish

## Focus
- Microcopy consistency
- Destructive-action language quality
- Typography hierarchy for secondary metadata
- Naming/capitalization consistency

## Implemented in this phase
- Model selector placeholder: `select model...` -> `Select model`
- Status metadata text increased from `text-xs` to `text-sm`
- Error/failed model status switched to semantic `text-destructive`
- Thinking status copy: `Thinking....` -> `Thinking…`
- Destructive confirmation copy:
  - `Are you absolutely sure?` -> `Delete all conversations?`
  - `This action cannot be undone...` -> `This action can’t be undone.`
- Branding copy consistency: `GitHub` capitalization
- Settings tab labels simplified:
  - `via Custom Host` -> `Custom Host`
  - `via Ollama Cloud` -> `Ollama Cloud`
