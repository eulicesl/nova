# N.O.V.A.

N.O.V.A. is a native-feeling AI chat app built to make local and cloud model conversations fast, clean, and genuinely pleasant to use.

This project is being developed as an independent product with its own identity, direction, and release path. The focus is simple:
- polished mobile UX
- fast chat flows
- strong local-first ergonomics
- a product story that feels intentional, not cobbled together

## What N.O.V.A. is for

N.O.V.A. is for people who want:
- a smooth mobile interface for chatting with AI models
- support for local-first workflows through Ollama
- a clean, minimal interaction model without unnecessary friction
- a product that treats responsiveness, readability, and quality as core features

## Current focus

Right now, the product is focused on:
- tightening identity and product coherence
- improving authorship clarity across app, repo, and release surfaces
- continuing to refine chat UX, performance, and platform polish

## Feature highlights

- **Native-quality interface** — clean, mobile-first chat UI
- **Zero-friction onboarding** — fast path into actual usage
- **Local-first privacy** — chats and settings stay on device
- **Thinking mode toggle** — quickly enable deeper reasoning where supported
- **Streaming replies** — visible response progress in real time
- **Dark mode everywhere** — polished appearance across themes
- **Rich Markdown rendering** — code, math, and structured output render cleanly
- **Persistent chat history** — jump back into prior conversations instantly
- **Live Activities** — follow ongoing chats from the home or lock screen
